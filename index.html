MS3: New foundations for micro-services and serverless systems

About

 The MS3 project aims at improving the foundations of micro-services and serverless computing, by introducing new consistency models that capture causal dependencies between user requests that are lost when traversing multiple heterogeneous services, and also by improving the Function as a Service (FaaS) model so that it fully realizes the serverless vision, namely by making it broadly applicable to different types of applications while also not requiring any type of resource management.

 

The MS3 project is funded by FCT (Fundação para a Ciência e a Tecnologia), under grant number PTDC/CCI-INF/6762/2020. The Principal Investigator leading the project is Rodrigo Miragaia Rodrigues. The host insitution is INESC-ID.
